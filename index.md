## About

Welcome to the Graph Theory GitHub website! This project is part of [coq-community](https://github.com/coq-community/manifesto).

This is a library of formalized graph theory results, including various
standard results from the literature (e.g., Menger’s Theorem, Hall’s
Marriage Theorem, and the excluded minor characterization of
treewidth-two graphs) as well as some more recent results arising
from the study of relation algebra within the ERC CoVeCe project
(e.g., soundness and completeness of an axiomatization of graph
isomorphism).

This is an open source project, licensed under the CeCILL-B.

Large parts of this library have been develped as part of the [ERC CoVeCe project](https://perso.ens-lyon.fr/damien.pous/covece/).
See also the [CoVeCe Graph-Theory project page](https://perso.ens-lyon.fr/damien.pous/covece/graphs/).
